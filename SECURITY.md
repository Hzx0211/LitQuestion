# Security Policy

## Supported Versions

LitQuestion is currently maintained on the latest `main` branch.

## Reporting a Vulnerability

If you discover a security issue:

- Do not open a public issue for anything involving API keys, credential leakage, arbitrary file access, or code execution.
- Use GitHub's private reporting flow if it is enabled for this repository.
- If private reporting is not available, contact the maintainer through the GitHub profile before disclosing details publicly.

Please include:

- What the issue is
- How it can be reproduced
- What impact you believe it has
- Any suggested mitigation, if known

I will aim to acknowledge reports quickly and follow up with a fix or mitigation plan.
